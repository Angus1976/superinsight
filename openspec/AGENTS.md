# AI Agents 责任一览

- AdminAgent：维护管理员后台、权限配置、告警逻辑
- AnnotationAgent：调度标注任务、工时采集、协作质量保障
- CorpusAgent：处理语料治理、归纳 BI 指标、反哺规则库
- RuleAgent：管理业务规则、智能推荐、规则校验引擎
- BillingAgent：汇总工时和账单、支持多币种与审批记录
- DeploymentAgent：监控混合部署状态、缓存一致、连接探活
- PerformanceAgent：追踪分片/归档表现、优化查询与缓存策略
- SecurityAgent：执行权限矩阵、脱敏、审计链与异常检测
- CoreAgent：支撑多租户核心能力、Superset 定制与中文优化
